### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 房屋信息
5. 我要当房主
6. 公告信息
7. 留言反馈
8. 个人中心
9. 后台管理（个人中心、预约看房、合同信息、房屋报修、维修处理、房屋评价、我要当房主和我的收藏管理）


**普通房主功能：**

6. 合同信息管理

8. 维修处理管理

**管理员功能：**
